What is Claude Platform?

- **Claude Platform is Anthropic's API service for building apps powered by Claude AI. It provides Python and TypeScript SDKs, tools for testing prompts, and guides for creating AI assistants and agents.**
- Claude Platform provides developers with API access to Anthropic's Claude AI models, including Opus, Sonnet, and Haiku variants, for building AI-powered applications and agents. The platform offers comprehensive SDKs in TypeScript and Python, prompt engineering tools, evaluation capabilities, agent skills for specialised tasks, and best practices for creating production-ready AI systems. Developers can leverage extended thinking capabilities, parallel tool execution, context management, and the Model Context Protocol (MCP) to build sophisticated agents that handle complex workflows, from coding assistants to business automation tools.
