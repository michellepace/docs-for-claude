What is Vercel?

- **Vercel hosts your websites and web apps in the cloud. Push your code to Git and Vercel automatically deploys it, giving you preview links for pull requests and a live production site.**
- Vercel is a cloud platform for deploying and hosting web applications with built-in Git integration and global edge infrastructure. It provides automatic deployments from Git commits, preview environments for every pull request, and production environments with custom domain support. The platform includes developer collaboration tools, analytics, deployment protection, and infrastructure services like serverless functions and CDN delivery optimised for frontend frameworks.
